# Steering: AI Notification Priority Classifier Rules

## 1. 분류 제약 조건 (Hard Rules)
- **긴급(URGENT)**: 마감 24시간 이내, 직속 상사/교수 요청, ASAP/마감/즉시 등의 키워드 포함.
- **오늘처리(TODAY)**: 마감 72시간 이내, 동료의 협업 요청, 미팅 조율.
- **참고용(REFERENCE)**: 전체 공지, 광고, FYI 성격의 정보 공유.

## 2. 맥락 추론 지침
- 단순 키워드가 아닌 **발신자 정보**를 최우선으로 고려할 것.
- 한국어의 **완곡한 표현** 뒷면에 숨겨진 긴급성을 파악할 것.
- 현재 시간(`current_time`)을 기준으로 상대적인 마감 임박 정도를 계산할 것.

## 3. 출력 형식 제약
- 보조 텍스트 없이 `[` 로 시작하고 `]` 로 끝나는 JSON 배열만 출력한다.
- 요약은 최대한 명사형 종결 어미를 사용하여 간결하게 처리한다.
