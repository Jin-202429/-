# Tasks: Execution Steps

## Step 1: 데이터 수집 및 전처리
- 사용자가 입력하거나 연결된 API로부터 메시지 원문(발신자, 제목, 본문, 시간)을 가져온다.

## Step 2: 컨텍스트 조합
- 시스템 프롬프트 + 현재 시각 + 사용자 신상 정보 + 메시지 리스트를 하나의 거대 프롬프트로 생성한다.

## Step 3: AI 추론 및 파싱
- Gemini API에 요청을 보내고, 반환된 텍스트에서 JSON 블록을 추출하여 객체화한다.

## Step 4: UI 렌더링
- 파싱된 결과를 우선순위별로 정렬하여 화면에 표시한다.
- '긴급' 항목은 시각적 애니메이션(Pulse)과 함께 최상단에 배치한다.

## Step 5: 사용자 피드백 처리
- 사용자가 '처리 완료'를 누르면 리스트에서 제거하거나 상태를 변경한다.
